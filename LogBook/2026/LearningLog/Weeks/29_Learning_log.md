##### Hoy aprendí

+ TDA -> Tipo de dato abstracto
+ Un mismo TDA puede implementarse de diferentes maneras concretas. 
+ Una estructura abstracta (TDA) se materializa en tu código a través de una estructura concreta.
+ **No existe una correspondencia uno a uno entre una estructura concreta y un TDA.**
+ **Una misma implementación puede servir para varios TDA y viceversa**
+ Pasos para elegir la mas eficiente y mejor opción entre las estructuras de datos según la necesidad.
